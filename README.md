# MistralBluff
A poker bot using mistral llm - Mistral AI Fine tune Hackathon

## Installation

```bash
pip install -r requirements.txt
```

## Usage

### Backend
```bash
source venv/Scripts/activate
cd backend
uvicorn main:app --reload
```

### Frontend
```bash
source venv/Scripts/activate
streamlit run frontend/poker_interface.py
```
